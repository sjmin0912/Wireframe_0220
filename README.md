# Wireframe_0220
# 🎮 Vibe Coding - AI를 활용한 게임 개발 학습 플랫폼

> **블록코딩 → 파이썬 전환 구간에서 이탈을 줄이기 위해 설계된 AI 기반 교육 플랫폼**  
> "딸깍(즉시 결과) → 산출물(통제 경험) → 자기화(확장 선택)"의 3단 구조로 학습자의 몰입과 성장을 동시에 확보합니다.

---


## 🎯 프로젝트 개요

### 배경
- **현 상황**: 블록코딩은 흥미가 높고 진입이 쉽지만, 파이썬/텍스트 코딩으로 전환할 때 **문법 장벽**으로 이탈 발생
- **문제**: "문법을 배우는 시간"이 길어질수록 학습자는 흥미를 잃고, 학습 사이클이 짧아짐
- **핵심 니즈** (학습자 관점)
  - "뭘 만들고 싶은데 쉽게 만들고 싶다"
  - "결과물을 빨리 보고 싶다 (딸깍 경험)"
  - "동시에 내가 뭘 배웠는지가 남아야 한다"

### 해결 방안
**"블록→파이썬 전환 구간"에서 이탈을 줄이기 위해, Vibe Coding으로 '빠른 산출물 + 사고/검증 루프'를 만들고 학습 사이클을 늘린다.**

---

### 학습자 성장 정의

| 성장 영역 | 설명 | 평가 기준 |
|----------|------|----------|
| **의도 명료화** | 내가 만들고 싶은 것을 1문장 목표로 표현 | ⭐⭐⭐⭐⭐ (매우 명확함) |
| **조건 설계** | "성공"을 기능/규칙으로 정의 | 3개 이상, "___하면 ___가 된다" 형식 |
| **테스트/검증** | 조건이 실제로 작동하는지 스스로 확인 | 3개 이상, 검증 가능함 |

*주차별로 평가 기준 개수 조절 가능 (n>=1 이상)
